## 安装说明
这是个油猴脚本，直接安装启用即可。

什么，竟然有人找不到脚本在哪里？当然是在 [GitHub Release](https://github.com/DuckSoft/91porn-utility/releases) 里面啦！


## 关于去广告功能
正常情况下浏览器由于需要加载大量的广告页面，所以页面加载完成事件的通知实际上相当缓慢，等待过程非常痛苦。

这时你只需要**双击页面**，脚本会强制执行自动去广告、挂载操作。
